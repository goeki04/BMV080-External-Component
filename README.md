ESPHome BMV080 Sensor ComponentThis repository contains an External Component for ESPHome to integrate the Bosch BMV080 Particulate Matter (PM) sensor via I2C. The implementation utilizes the official Bosch Sensortec BMV080 sensor driver.FeaturesSupport for PM2.5, PM1.0, and PM10 mass concentration ($\mu g/m^3$).Support for PM2.5, PM1.0, and PM10 number concentration (particles/$cm^3$).Obstruction detection flag to indicate if the sensor field of view is blocked.Continuous Measurement Mode for real-time air quality monitoring.PrerequisitesTo compile and flash this project, you must have the following installed locally on your machine:Python 3.x: Required to run the ESPHome build system.ESPHome (Local Installation): It is recommended to install ESPHome in a virtual environment:Bash
